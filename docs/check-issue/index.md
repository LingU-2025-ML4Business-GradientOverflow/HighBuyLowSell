# Check-Issue 文档索引

本目录用于把当前项目的技术路线、分工逻辑和每个阶段的任务包写清楚，避免出现以下问题：

1. 大家都在做事，但不知道彼此在做什么。
2. 每个人都写了一点代码，但最后拼不起来。
3. 任务名看起来很清楚，实际交付物并不清楚。
4. 报告和代码脱节，最后只能靠临时拼接。
